# Turtle Crossing Game

Help the turtle safely cross the road while avoiding moving cars in this fun arcade-style game built with Python's **Turtle** graphics module. Each successful crossing increases the game's difficulty, making it more challenging to reach the finish line!

## 🎯 About the Project

In this game, you control a turtle that must cross a busy road filled with moving cars. Every time the turtle reaches the finish line, the level increases and the cars move faster. The game ends if the turtle collides with a car.

## ✨ Features

- 🐢 Control the turtle using the **Up Arrow** key
- 🚗 Randomly generated moving cars
- 📈 Increasing difficulty with each level
- 🏆 Level tracking using a scoreboard
- 💥 Collision detection and Game Over screen
- 🎨 Built with Python's Turtle graphics
